# Dev Container Features

Мой набор Features для Dev Container.

Все тесты на образе `mcr.microsoft.com/vscode/devcontainers/base:ubuntu`

Ручной запуск тестов:

```sh
# Запуск в указанном image
devcontainer features test --features <имя фичи> --base-image mcr.microsoft.com/vscode/devcontainers/base:ubuntu

# Запуск сценариев
devcontainer features test --skip-autogenerated .

devcontainer features test --features opencode --skip-autogenerated .
devcontainer features test --features ansible-molecule --skip-autogenerated .
```
